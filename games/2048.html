2048

Merge identical numbers to create a 2048 block! Use arrow keys to move the blocks, and a new 2 or 4 block will appear after each move.

Game Rules

  • Use the arrow keys (up, down, left, right) to move the blocks.
  • On mobile devices, use the on-screen direction buttons below.
  • When two blocks with the same number collide, they will merge into a new block, and the value is the sum of the two blocks.
  • After each move, a new 2 or 4 will appear in an empty space.
  • When you reach a 2048 block, you win! But you can continue to play for a higher score.
  • When there is no space to move, the game ends.
  • The score is calculated based on the blocks you merge. The value of the new merged block will be added to your score.
  • Use the "Undo" button to return to the previous state.
